# Система классификации состояния глаз

Проект разделен на три основных файла для удобного разделения обучения и инференса.

## Структура проекта

- **`train.py`** - Обучение модели
- **`infer.py`** - Инференс (предсказания) без переобучения  
- **`main_fast.py`** - Универсальный запуск с выбором режима
- **`images/`** - Папка с датасетами (zip файлы)
- **`lol.xml`** - Каскадный классификатор для детекции глаз

## Использование

### 1. Обучение модели
```bash
python train.py
```
или
```bash
python main_fast.py --mode train
```

### 2. Инференс с веб-камеры
```bash
python infer.py --webcam
```
или
```bash
python main_fast.py --mode infer
```

### 3. Анализ изображения
```bash
python infer.py --image path/to/image.jpg
```

### 4. Интерактивный режим
```bash
python main_fast.py
```

## Файлы модели

После обучения создаются файлы:
- `eye_classifier.pkl` - обученная модель
- `eye_scaler.pkl` - нормализатор данных
- `class_mapping.pkl` - маппинг классов

## Классы

Модель различает 3 состояния глаз:
- **Класс 0**: Close Eyes (закрытые глаза)
- **Класс 1**: Drowsy (сонные глаза) 
- **Класс 2**: Open Eyes (открытые глаза)

## Требования

- Python 3.7+
- OpenCV
- scikit-learn
- numpy
- joblib
- tqdm

## Особенности

- Автоматическое обнаружение и обработка zip архивов в папке `images`
- Поддержка различных форматов изображений (jpg, png, bmp)
- Оптимизированная детекция глаз для разных разрешений
- Интерактивный режим с веб-камерой
- Возможность анализа отдельных изображений
