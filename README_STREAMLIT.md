# 😴 Детекция Сонливости - Streamlit Приложение

Это приложение использует обученную модель машинного обучения для детекции сонливости в реальном времени через веб-камеру.

## 🚀 Быстрый запуск

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Обучение модели (если еще не обучена)
```bash
python train.py
```

### 3. Запуск приложения
```bash
python run_app.py
```

Или напрямую через Streamlit:
```bash
streamlit run app.py
```

## 📋 Требования

- Python 3.7+
- Веб-камера
- Файлы модели (создаются после обучения):
  - `eye_classifier.pkl`
  - `eye_scaler.pkl` 
  - `class_mapping.pkl`
  - `lol.xml` (каскадный классификатор)

## 🎯 Функциональность

### Основные возможности:
- **Реальное время**: Детекция через веб-камеру
- **Простой интерфейс**: Два статуса - "Спит" или "Не Спит"
- **Визуальная обратная связь**: Цветные рамки вокруг глаз
- **Статистика**: Уверенность модели и количество обнаруженных глаз

### Статусы детекции:
- 🟢 **"Не Спит"** - глаза открыты и активны
- 🔴 **"Спит"** - глаза закрыты или сонные

## 🖥️ Интерфейс

### Левая панель - Видео:
- Кнопка "Запустить камеру"
- Кнопка "Остановить камеру" 
- Живое видео с веб-камеры
- Рамки вокруг обнаруженных глаз

### Правая панель - Статус:
- Текущий статус (Спит/Не Спит)
- Уровень уверенности модели
- Количество обнаруженных глаз
- Инструкции по использованию
- Информация о модели

## ⚙️ Настройки

Приложение автоматически адаптируется к разрешению камеры:
- **HD+ (1920px+)**: Более точная детекция
- **HD (1280px)**: Стандартные параметры
- **SD (<1280px)**: Оптимизированные параметры

## 🔧 Технические детали

### Модель:
- **Алгоритм**: Logistic Regression
- **Входные данные**: 32x32 пикселя (серые изображения глаз)
- **Классы**: 3 (закрытые, сонные, открытые глаза)
- **Препроцессинг**: StandardScaler

### Детекция глаз:
- **Каскадный классификатор**: Haar Cascades
- **Адаптивные параметры**: В зависимости от разрешения
- **Многомасштабная детекция**: Разные размеры окон

## 🐛 Решение проблем

### Камера не работает:
1. Проверьте подключение веб-камеры
2. Убедитесь, что камера не используется другим приложением
3. Проверьте права доступа к камере

### Модель не загружается:
1. Убедитесь, что файлы модели существуют
2. Запустите обучение: `python train.py`
3. Проверьте права доступа к файлам

### Низкая точность детекции:
1. Улучшите освещение
2. Расположитесь лицом к камере
3. Убедитесь, что глаза видны четко

## 📱 Использование

1. **Запустите приложение** через `python run_app.py`
2. **Откройте браузер** по адресу http://localhost:8501
3. **Нажмите "Запустить камеру"**
4. **Смотрите прямо в камеру**
5. **Следите за статусом** в правой панели
6. **Остановите камеру** когда закончите

## 🎨 Кастомизация

Вы можете изменить:
- **Пороги уверенности** в функции `get_drowsiness_status()`
- **Цвета рамок** в основном цикле обработки видео
- **Параметры детекции глаз** в функции `detect_eyes()`
- **Интерфейс** через Streamlit компоненты

## 📊 Мониторинг

Приложение показывает:
- **Статус в реальном времени**: Спит/Не Спит
- **Уверенность модели**: От 0 до 1
- **Количество глаз**: Обнаруженные глаза
- **Информацию о модели**: Классы и их ID
